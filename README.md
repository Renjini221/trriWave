# trriWave
TriWave is a custom-built active audio system engineered for powerful, precise, and immersive sound reproduction. Designed around a 3-way active crossover architecture, the system separates low, mid, and high frequencies for dedicated amplification, delivering cleaner output, improved clarity, and deeper bass response.
